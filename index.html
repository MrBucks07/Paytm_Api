Mr Bucks Web Services!

This is an Api to generate Paytm Transaction Token

------------------------------

Simply you need to send a request on end point : /genTransactionToken

Required parameters in body (Json)
----------------------------------
1 : mid (merchant id)
2 : keySecret
3 : orderId
4 : callBackUrl (same as requesting url (copy and paste))
5 : websiteName (WEBSTAGING)
6 : txnAmount (string not int)
7 : custId
